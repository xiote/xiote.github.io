# CLI CODE GENERATROR

## CLAUDE-CODE

```
$ brew install claude-code

$ claude --version
2.0.33 (Claude Code)

# 기본설정, https://code.claude.com/docs/ko/settings
# ~/.claude/settings.json
{
  "model": "Sonnet"
}

# yolo모드 실행, 설정불가
$ claude --dangerously-skip-permissions

# 기본지침
# ~/.claude/CLAUDE.md, (~/.claude/CLAUDE.md에 들어갈 지침이야. 영어로 바꿔줘.)
-	After adding a feature, always run the corresponding tests.
-	Once testing is complete, terminate any related web servers.
-	Always document dependencies in the README.md file.

# 우분투 설치
$ curl -fsSL https://claude.ai/install.sh | bash
```

## GEMINI

```
$ brew install gemini-cli

$ gemini --version
0.12.0

# 기본설정
# ~/.gemini/settings.json
{
  "security": {
    "auth": {
      "selectedType": "oauth-personal"
    }
  },
  "model": {
    "name": "gemini-2.5-pro"
  }
}

# yolo모드 실행, 설정불가
$ gemini --yolo

```

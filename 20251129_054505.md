# 인터파크 좌석미리보기

## Onestop 공연 좌석 미리보기

### 1. 기본 실행 명령

```bash
echo "https://tickets.interpark.com/onestop/api/seats/block-data?goodsCode={goodsCode}&placeCode={placeCode}&playSeq={playSeq}" | python3 ./onestop-seats/generateSeatMap.py > onestopTemp.html
open -a "Google Chrome" onestopTemp.html
```

-   `{goodsCode}` : 공연 상품 코드
-   `{placeCode}` : 공연장(공연 장소) 코드
-   `{playSeq}` : 회차 번호

위 값들을 실제 공연 정보에 맞게 치환하여 사용하시면 됩니다.

---

### 2. 예시

아래는 실제 값이 들어간 예시입니다.

```bash
echo "https://tickets.interpark.com/onestop/api/seats/block-data?goodsCode=25013370&placeCode=21000396&playSeq=083" | python3 ./onestop-seats/generateSeatMap.py > onestopTemp.html
open -a "Google Chrome" onestopTemp.html
```

위 명령을 실행하면:

1. 인터파크 Onestop 좌석 API에서 해당 공연의 좌석 블록 데이터를 조회하고,
2. `generateSeatMap.py` 스크립트가 이 데이터를 HTML 구조로 변환하여 `onestopTemp.html` 파일로 저장하며,
3. `onestopTemp.html`을 **Google Chrome**으로 자동 실행하여 좌석 배치를 미리 확인하실 수 있습니다.

---

> 참고
> 위의 모든 명령은 setMo 프로젝트를 git clone 한 뒤, 해당 프로젝트 경로(예: setMo 또는 setMo/onestop-seats)에서 실행해야 정상적으로 동작합니다.
